# StockSikh 📈

StockSikh is an AI-powered stock market learning platform designed for beginners. It helps users understand stocks through simple explanations, technical indicators, real-time market data, and an intelligent AI mentor called Vidya AI.

## Features

### 📊 Real-Time Stock Analysis

* Live stock prices
* Interactive price charts
* RSI (Relative Strength Index)
* Moving Averages (MA20 & MA50)
* BUY / SELL / HOLD signals

### 🤖 Vidya AI Assistant

* Powered by Google Gemini AI
* Explains stock concepts in simple language
* Interprets technical indicators
* Provides educational market insights

### ⭐ Watchlist Management

* Add and remove stocks
* Track favorite stocks
* Monitor signals and price movements

### 🔍 Stock Discovery

* Search Indian stocks
* View company details
* Explore market trends

### 📰 Market News

* Latest stock market news
* Indian market updates
* News-driven learning experience

### 🔐 User Authentication

* Secure login and signup
* Personalized dashboard
* User-specific watchlists

## Tech Stack

### Frontend

* React.js
* Tailwind CSS
* Axios
* Recharts

### Backend

* FastAPI
* SQLAlchemy
* MySQL
* Yahoo Finance API

### AI

* Google Gemini API

## Goal

StockSikh aims to simplify stock market education by combining real-time financial data with AI-powered explanations, making investing concepts easier for beginners to understand.
